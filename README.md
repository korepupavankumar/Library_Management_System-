# Library Management System

## Objective

The objective of this project is to automate the management of library books using Python.

## Features

- Add Book
- View Books
- Search Book
- Issue Book
- Return Book

## Technologies Used

- Python
- File Handling

## Files Used

- library.py
- books.txt
- issued_books.txt

## How to Run

python library.py

## Conclusion

This project helps manage library books efficiently and reduces manual work.
